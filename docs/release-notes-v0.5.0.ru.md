# Planfix Field Exporter 0.5.0

## Главное

Релиз добавляет поиск использований выбранного поля в формулах Planfix через
локальный индекс.

Теперь приложение умеет:

- строить локальный индекс формул;
- искать выбранное поле в вычисляемых полях задач и контактов;
- искать выбранное поле в текстовых полях записей справочников;
- учитывать ссылки внутри `{{...}}`, включая пути вида
  `{{Задача.Сделка / Название поля}}`;
- сохранять результаты поиска в CSV/JSON в выбранную папку выгрузок.

## Что изменилось в интерфейсе

- Добавлена кнопка `Обновить индекс`.
- Кнопка `Найти в формулах` теперь ищет по локальному индексу без HTTP-запросов
  к Planfix.
- Добавлено окно выбора справочников для поиска формул.
- Добавлена кнопка `Остановить` для безопасной остановки долгой операции.
- В строке состояния показывается текущий этап, прогресс, последний запрос и
  путь output-файла.
- В окне результатов поиска выводится название записи справочника.
- Ширину столбцов таблицы можно менять мышью.

## Надёжность

- Между любыми HTTP-запросами к Planfix выдерживается пауза не менее `1.25`
  секунды.
- При временных сетевых ошибках и HTTP 500/502/503/504 приложение повторяет
  запрос.
- Если отдельный справочник не удалось прочитать, приложение старается
  продолжить обновление индекса и сохраняет подробности в `index_errors.json`.
- При остановке долгой операции частичные результаты сохраняются.

## Где лежат файлы

- Локальный индекс:
  `<Папка выгрузок>\formula_index`
- Результаты поиска:
  `<Папка выгрузок>\formula_search\<дата_и_время>`
